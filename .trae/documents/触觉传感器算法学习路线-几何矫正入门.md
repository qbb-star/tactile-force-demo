## 三、Phase 1：几何矫正 —— 详细学习计划

### 3.1 核心问题

触觉传感器相机拍摄的原始图像存在两个问题：
1. **镜头畸变**：广角/微距镜头带来的桶形/枕形畸变
2. **透视偏差**：相机与凝胶表面不平行，导致点阵非等距、非正交

几何矫正的目标：**将原始图像中的非均匀点阵，映射到标准化的均匀网格上**。

### 3.2 底层原理（逐步讲解）

#### Step 1: 理解"标记点阵"的作用

你的触觉图像上有一层印制/喷涂的标记点（marker dots），它们的作用是：
- 提供**空间参考坐标系**（类似标定板的棋盘格角点）
- 物理间距已知（pitch_x_mm = pitch_y_mm = 0.6mm），提供**尺度信息**
- 点阵的位移 = 凝胶表面的形变，这是后续力/形变分析的基础

#### Step 2: DoG（高斯差分）检测原理

标记点是暗色圆点（在亮背景下），DoG 的工作原理是：

```
1. 大核高斯模糊 → 保留低频背景
2. 小核高斯模糊 → 保留更多细节（包括标记点）
3. 两者相减 → 突出与背景有差异的区域（即标记点）
4. 乘以增益 → 增强对比度
5. OpenCV SimpleBlobDetector → 检测圆形 blob
```

对应代码在 `markers.py` 的 `_preprocess_dog()` 函数。

#### Step 3: PCA 网格排序原理

检测到的标记点是**无序的点集**。需要把它们排成规则的 (row, col) 网格：

```
1. 计算点集的协方差矩阵，做特征分解 (PCA)
2. 第一主成分 ≈ 列方向，第二主成分 ≈ 行方向
3. 将点投影到这两个主轴上
4. 用 k-means 聚类分别对行/列投影坐标分组
5. 贪心分配：每个点分配到最近的 (row, col) 格子
```

对应代码在 `markers.py` 的 `index_points_to_grid()` 函数。

#### Step 4: 透视矫正（Homography）原理

四个角点确定后，可以计算透视变换矩阵 H：

```python
# 源点：四个角在原始图像中的位置
src = [tl, tr, br, bl]  # 原始图像坐标
# 目标点：四个角在标准网格中的位置
dst = [(0,0), (n_cols-1,0), (n_cols-1,n_rows-1), (0,n_rows-1)]
# 计算 3×3 透视变换矩阵
H = cv2.getPerspectiveTransform(src, dst)
```

#### Step 5: 逐三角剖分的仿射映射（warp 构建）

因为镜头畸变是非线性的，单一 Homography 不够。viewtacv2 使用**逐网格单元的双线性映射**：

```
1. 将原始网格点扩展一圈（边界外推）
2. 生成标准网格（canonical grid），保持物理纵横比
3. 每个四边形单元拆分为两个三角形
4. 对每个三角形计算仿射变换 (cv2.getAffineTransform)
5. 遍历标准网格的每个像素，通过所属三角形的仿射变换找到原始图像中的对应位置
6. 生成稠密的 map_x, map_y 查找表
```

对应代码在 `warp.py` 的 `dense_map_from_grids()` 函数。

数学表达：
$$\mathbf{I}_r(x,y)=\mathbf{I}(W_x(x,y),W_y(x,y))$$

其中 $W_x, W_y$ 是 warp 的两个坐标通道（即 map_x, map_y）。
